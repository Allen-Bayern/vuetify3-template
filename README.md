# Vuetify 3 模板

## 使用方法

1. 将本库`clone`之后，首先`git remote remove origin`。然后换成你自己的源。
2. 必要的话，删除`package-lock.json`，换成自定义的包管理器`npm`, `yarn`, `pnpm`等。如换`pnpm`，需注意`node >= 16`。

## 命令

**开发**

```sh
#npm
npm run dev

# yarn
yarn dev

# pnpm
pnpm dev
```


**构建**

```sh
#npm
npm run build

# yarn
yarn build

# pnpm
pnpm build
```
